# Документация ITS

**ITS (Intelligent Trading Strategies)** - интеллектуальная система формирования, тестирования и эволюционной генерации торговых стратегий для работы с биржевыми данными фондового, валютного и криптовалютного рынков.

Основной пользователь системы - финансовый модельер, квант-исследователь или аналитик, который проектирует торговые модели, проверяет их устойчивость на исторических данных и собирает новые стратегии из стандартизированных компонентов.

## Быстрый старт

1. Установить Docker и Docker Compose v2.
2. Создать файл `.env` в корне проекта и указать токен T-Invest:

```dotenv
tinvest_token=your_tinkoff_invest_token
```

3. Запустить систему одной командой:

```bash
docker compose up --build
```

4. Открыть главное окно:

[http://localhost:8080/launchpad/](http://localhost:8080/launchpad/)

## Что получает пользователь

Система дает финансовому модельеру единое рабочее пространство, в котором можно:

- подключать источники рыночных данных;
- исследовать инструменты, котировки, дивиденды и производные бары;
- создавать компоненты торговой модели: `pre-selection`, `signal`, `allocation`;
- собирать стандартизированные стратегии из этих компонентов;
- тестировать стратегии через CPCV, WalkForward и Backtesting;
- сравнивать стратегии по агрегированному рейтингу;
- задавать алфавиты генетического алгоритма и автоматически генерировать новые стратегии;
- материализовывать лучшие GA-кандидаты в Python-код в реестре моделей.

## Статус текущей реализации

Текущая версия является исследовательско-лабораторной платформой для формирования, анализа, тестирования и генерации торговых стратегий. Система содержит инфраструктуру данных, UI, API, ядро стратегий, тестирование и GA-генератор. Промышленное исполнение заявок на брокерском счете, риск-контроль production-контуров и модуль управления ордерами могут быть реализованы как отдельные расширения.
