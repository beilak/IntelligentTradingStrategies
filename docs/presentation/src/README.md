# ITS reveal.js presentation

Готовые файлы презентации лежат в этой папке.

## Как использовать

1. Скопируйте содержимое `docs/presentation/src` в нужную папку презентации.
2. В текущей версии `index.html` ожидает reveal.js на два уровня выше:
   - `../../dist/reset.css`
   - `../../dist/reveal.css`
   - `../../dist/reveal.js`
3. Откройте `index.html` через локальный сервер reveal.js или любым способом, который уже используется для запуска reveal.js.

## Состав

- `index.html` - сама презентация.
- `theme/its-bank.css` - премиальная темная тема в стиле инвестиционного продукта.
- `assets/img` - локальные копии скриншотов из `docs/img`.

Презентация не требует установки дополнительных пакетов и не подтягивает внешние шрифты или CDN.
